📖 GAME RULES

掌握這些按鍵，你才能在 BRICKADE 世界隨心所欲。

← → 左右移動
↓ 軟降 Soft Drop (+1 分 / 格)
Space 硬降 Hard Drop (+2 分 / 格)
↑ / X 順時針旋轉
Z / Ctrl 逆時針旋轉
C / Shift 保留 Hold (每顆限用一次)
P / Esc 暫停 / 繼續
R 重新開始 / 對戰後離開房間
A Undo 反悔上一步
M / N 音樂 / 音效靜音
+ / - 遊戲音量大 / 小
1 ~ 4 發送表情 (對戰)
F ×2 連按兩下投降 (對戰中)
💡 Tip：Hold (C) 是高手的必備工具，存下 I 型方塊等大招會讓你火力翻倍！

單人模式的核心是 消行 → 得分 → 升級 → 變快。越高級分數倍率越高，但速度也越恐怖。

🛡️ 垃圾抵銷 & 寬限期

對手送來的垃圾會先以 🟡 黃色警告條 停留 2 秒，這段期間只要你成功消行，攻擊可以反打抵銷！
時間一過就變成 🔴 紅色危險條，下次落地就會從盤底真的頂上來。

抵銷順序：你的攻擊 → 先抵自己的 🔴 紅色 → 再抵 🟡 黃色 → 有剩才送給對手。

💀 KO 判定

✅ 被 KO 可復活
頂到天花板時，盤上有垃圾行 → 被 KO 但復活，對手 +1 KO，比賽繼續。
❌ TOP OUT 直接輸
頂到天花板時，盤上沒有垃圾行（是你自己堆上去的） → 直接落敗。
🤝 死亡寬限期：雙方在 300ms 內幾乎同時死亡 → 判定 平手（DRAW），吸收網路延遲造成的「搶先一步」不公平。

🏅 勝負 & 段位

時間結束時，KO 較多者獲勝；KO 相同則比較 LINES SENT。贏了加 LP 爬段位，輸了扣 LP。
