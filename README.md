# BRICKADE

A real-time online block-stacking battle game playable in any modern browser.
Self-hosted, P2P multiplayer, no install required.

## Features

- 1v1 real-time online battles via WebRTC (PeerJS)
- AI opponents across 5 difficulty tiers (rookie → god) powered by a C++ engine compiled to WebAssembly
- Two battle modes: **BOMB** (signature mode — garbage rows are explosive bombs) and **CLASSIC**
- Modern competitive ruleset: SRS rotation, 7-bag randomizer, hold piece, ghost piece, T-spin, B2B, Perfect Clear
- Combo Room and Free Placement training modes
- Ranked LP / tier system with cloud profiles (Firebase)
- PWA-installable, offline single-player support

## Tech stack

- Vanilla JS, HTML5 Canvas 2D
- C++ AI compiled via Emscripten → WebAssembly, run inside a Web Worker
- Firebase (Auth + Firestore + Realtime DB) for accounts, matchmaking, stats
- PeerJS for game-state P2P sync
- Service Worker for offline / PWA install

## Development

```sh
# Run locally — just serve src/ with any static server
npx serve src

# Build & deploy to Firebase Hosting (obfuscates game.js + ai_worker.js)
npm run deploy
```

The `public/` directory is generated by `build.js`; do not edit it directly.
Always edit files inside `src/`.
